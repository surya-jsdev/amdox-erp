# 🚀 Amdox ERP System

Amdox ERP System is a modern Enterprise Resource Planning (ERP) web application built using the MERN Stack. It helps organizations manage multiple business operations from a single platform, including Finance, HR, Inventory, Sales, Purchase, Projects, CRM, and Administration.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/amdox-erp.git
```

---

## Install Frontend

```bash
cd client
npm install
```

---

## Install Backend

```bash
cd server
npm install
```
